🗃️ Custom Columns

Add extra data fields (SKU, Supplier, Location, HSN Code, etc.) that appear as columns in Inventory and Products tables. Data is saved per product.

📌 How it works

  • Create a column once — it appears for all products in the chosen table(s).
  • Go to Inventory tab to fill in values for each product inline.
  • Go to Products tab to see the columns alongside standard product data.
  • Column types: Text, Number, Dropdown, Date.
  • Deleting a column removes its data from all products permanently.

🧩 Push to ShopHere.in — Chrome Extension

Browse any supplier website and push products directly to your store with one click. Works on DropshipIndia, CJ Dropshipping, Hubbazaar, AliExpress, Amazon, Flipkart and any other product page.

✨ How it works

🔌
1. Install Extension
Load the extension folder into Chrome in Developer mode
⚙️
2. Configure Once
Enter your store URL and push token in the extension popup
🛍️
3. Browse Supplier
Open any product page on any supplier website
🚀
4. Push to Store
Click the extension icon, set markup, click Push — done!
1

Set a Push Token (security key)

Go to Store Settings → Chrome Extension section → set a Push Token (any secret word). This prevents unauthorized pushes to your store.

2

Install the Extension in Chrome

  1. Open Chrome browser on your computer
  2. Go to chrome://extensions in the address bar
  3. Turn on Developer mode (toggle in top-right corner)
  4. Click "Load unpacked" button
  5. Navigate to your project folder: Kiro 1 → extension
  6. Select the extension folder and click OK
  7. The "Push to ShopHere.in" extension icon will appear in Chrome toolbar
💡 Extension folder location: C:\Users\Yadadri Manufacturer\Desktop\Kiro 1\extension
3

Configure Extension — Store URL & Token

  1. Click the "Push to ShopHere.in" extension icon in Chrome toolbar
  2. Click the ⚙️ Settings icon in the popup
  3. Enter your Store URL: https://shophere.in
  4. Enter your Push Token (same one you set in Step 1)
  5. Click Save & Connect — it will test the connection
4

Browse Any Supplier & Push Products

  1. Go to any supplier website — DropshipIndia, CJ Dropshipping, Hubbazaar, AliExpress, Amazon, Flipkart etc.
  2. Open a specific product page (not category or home page)
  3. Click the 🧩 Push to ShopHere.in extension icon
  4. It auto-scrapes: product name, price, images, description
  5. Set your Markup % — your selling price is calculated automatically
  6. Edit name, category, stock as needed
  7. Click 🚀 Push to ShopHere.in — product is added instantly!
✅ Supported suppliers: DropshipIndia · CJ Dropshipping · Hubbazaar · AliExpress · Amazon.in · Flipkart · Any Shopify store · Any WooCommerce store · and any other product page on the web
